# Tailwind CSS Guide

- **Tailwind CSS**는 신속하고 효율적인 UI 개발을 지원하는 유틸리티 기반 CSS 프레임워크입니다.
- **Just-in-Time Mode**를 통해 불필요한 CSS를 제거하고 필요한 스타일만을 빠르게 생성할 수 있습니다.
- 다크 모드, 반응형 디자인, 애니메이션 등 Tailwind CSS가 제공하는 다양한 기능을 활용하여 프로젝트 개발을 더욱 간단하고 효과적으로 만들 수 있습니다.

---

## 주요 기능

---

### 상태 기반 스타일링

#### **Hover, Focus, 및 기타 상태**

사용자의 상호작용에 따라 스타일을 지정할 수 있는 유틸리티 클래스를 제공합니다. 버튼에 마우스를 올리거나 입력 필드에 포커스가 있을 때, 손쉽게 스타일을 변경할 수 있습니다.

**예시**:

```tsx
<button class="bg-blue-500 hover:bg-blue-700 focus:ring focus:ring-blue-300">
  Click Me
</button>
```

##### **개선된 예시**

`active`나 `disabled` 상태를 추가해 대화형 요소를 더 풍부하게 만들 수 있습니다.

```tsx
<button class="bg-gray-300 active:bg-gray-500 disabled:opacity-50" disabled>
  Disabled Button
</button>
```

> 자세한 내용은 [Tailwind CSS 공식 문서](https://tailwindcss.com/docs/hover-focus-and-other-states)를 참조하세요.

---

### 형제 상태에 따른 스타일링 (`peer-{modifier}`)

`peer` 클래스를 사용해 특정 요소의 형제 상태에 따라 스타일을 지정할 수 있습니다. React와 같은 프레임워크에서 폼 유효성 검사에 유용합니다.

**예시**:

```tsx
<label>
  <span class="peer-invalid:text-red-500">Email</span>
  <input type="email" class="peer" />
</label>
```

> 참고: `peer`는 **이전 형제 요소**에만 적용됩니다.  
[공식 문서 바로가기](https://tailwindcss.com/docs/hover-focus-and-other-states#styling-based-on-parent-state)

## 반응형 디자인 _(Responsive Design)_

Tailwind CSS는 간단한 클래스 이름을 사용하여 미디어 쿼리를 쉽게 적용할 수 있습니다.

### **주요 Breakpoints**

| 키워드  | 최대 넓이 |
|---------|-----------|
| `sm`    | 640px     |
| `md`    | 768px     |
| `lg`    | 1024px    |
| `xl`    | 1280px    |
| `2xl`   | 1536px    |

**예시**:

```tsx
<div class="text-sm sm:text-md lg:text-lg">반응형 텍스트</div>
```

#### **커스텀 Breakpoints**

```javascript
module.exports = {
  theme: {
    screens: {
      "3xl": "1920px",
    },
  },
};
```

> [반응형 디자인 문서 보기](https://tailwindcss.com/docs/responsive-design)

## Dark Mode

Tailwind CSS는 다크 모드를 쉽게 구현할 수 있습니다. 사용자 환경 또는 선택에 따라 다크 테마를 동적으로 전환합니다.

### **다크 모드 활성화**

`dark` 클래스를 사용하여 다크 모드 스타일을 정의합니다.

```tsx
<div class="bg-white dark:bg-gray-800">
  <p class="text-black dark:text-white">Hello, Dark Mode!</p>
</div>
```

#### **다크 모드 동적 전환**

```javascript
document.documentElement.classList.toggle("dark");
```

`tailwind.config.js`에서 설정을 변경하여 트리거 방식을 수정할 수 있습니다.

```javascript
module.exports = {
  darkMode: "class", // 또는 'media'
};
```

> [Dark Mode 설정 문서](https://tailwindcss.com/docs/dark-mode)

## 애니메이션

Tailwind CSS는 기본 애니메이션 클래스를 제공하며, 복잡한 애니메이션은 커스텀 설정으로 확장할 수 있습니다.

### **기본 제공 클래스**

- `animate-spin`: 회전 효과
- `animate-bounce`: 상하 이동 효과

**예시**:

```tsx
<div class="animate-bounce">Bouncing Element</div>
```

##### **성능 최적화**

복잡한 애니메이션은 `will-change` 속성을 추가하여 성능을 개선하세요.

```css
.custom-animation {
  will-change: transform, opacity;
}
```

## 플러그인

Tailwind는 다양한 플러그인을 통해 기능을 확장할 수 있습니다. 그중 Forms 플러그인은 폼 요소의 스타일을 쉽게 관리할 수 있게 도와줍니다.

### **Forms 플러그인 설치 및 설정**

```bash
npm install -D @tailwindcss/forms
```

**활용 예시**:

```tsx
<input type="text" class="form-input" />
```

```javascript
require("@tailwindcss/forms")({
  strategy: "class", // 기본 스타일을 비활성화하고 클래스 기반으로 사용
});
```

> [Forms 플러그인 문서 보기](https://tailwindcss-forms.vercel.app)

## Just-in-Time Mode

JIT 모드는 실시간으로 필요한 CSS를 생성하여 성능을 최적화합니다.

### **활성화 방법**

`tailwind.config.js`에서 아래와 같이 설정합니다.

```javascript
module.exports = {
  mode: "jit",
};
```

**예시**:

```tsx
<div class="p-[3rem] text-[#123456]">Custom Spacing & Color</div>
```

## IntelliSense _(자동 완성)_

**Tailwind CSS IntelliSense** 확장 프로그램은 생산성을 높이는 유용한 도구입니다.

**설치 방법**:

[VS Code 확장 페이지](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss)에서 설치합니다.

**추가 설정**:

`settings.json` 파일에 아래 옵션을 추가하여 자동 완성을 활성화하세요.

```json
"editor.quickSuggestions": {
  "strings": true
}
```
